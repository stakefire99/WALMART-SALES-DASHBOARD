# 🛒 Walmart Sales Dashboard – Power BI

## 📌 Overview

This project presents an interactive Power BI dashboard analyzing Walmart sales data to uncover trends, patterns, and business insights.

---

## 🧠 Objective

* Analyze sales performance across regions and categories
* Identify top-performing products
* Track revenue trends over time

---

## 🛠️ Tools Used

* Power BI
* Data Modeling
* Data Visualization

---

## 📊 Key Insights

* Identified top-performing product categories and regions
* Highlighted seasonal and time-based sales trends
* Provided a clear view of revenue distribution and performance

---

## 📁 Files Included

* Power BI Dashboard (.pbix)
* Project Report / Screenshots

---

## 🚀 Outcome

The dashboard enables quick, visual understanding of sales performance, helping in better business decision-making.
